# TS_TL (TypeScript To-do-List in CLI)

a simple to-do list app using TypeScript for exercise

## install (Linux)

> git clone https://github.com/NorouziMilad/TS_TL.git

## start

Go to the folder `TS_TL` and use `npm start` to start and You should see this menu in your terminal:

```
Options Managers (Use arrow k eys)
❯ Managers
  Add new manager
  Modify manager
  Delete one manager
  Delete all managers
  Quit
```

After you create a manager, all data is saved in a file called `db.json`.
